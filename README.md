# logspam
tails (-f) all files found under /var/log

# Installation

copy this script to your home directory
and call it 

    ./logspam.sh
  

*Ctrl-C* to abort

*Ctrl-Z* to stop, use *bg* for sendig it to background


you also can call it with *&* at the end to start into background:

    ./logspam.sh &
